# 💈 Old School Barber

Um sistema web completo e responsivo para gerenciamento de agendamentos de uma barbearia com temática clássica/retrô. O projeto permite que clientes se cadastrem, façam login e agendem serviços com seus barbeiros favoritos de forma dinâmica.

##  Funcionalidades

* **Autenticação de Usuários:** Sistema de Cadastro, Login e Logout seguro.
* **Agendamento Dinâmico:** Seleção de profissionais, serviços, e horários disponíveis em tempo real.
* **Painel do Cliente:** Visualização dos próximos agendamentos.
* **Cancelamento:** Opção para o cliente cancelar agendamentos diretamente pelo painel.
* **Interface Retrô:** Design exclusivo focado na experiência do usuário, com responsividade para dispositivos móveis e desktop.

##  Tecnologias Utilizadas

* **Back-end:** Python, Flask, Flask-Login, Flask-SQLAlchemy
* **Front-end:** HTML5, CSS3, JavaScript (Vanilla)
* **Banco de Dados:** SQLite
* **Deploy:** Gunicorn, Render
